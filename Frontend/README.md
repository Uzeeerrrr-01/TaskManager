# 📝 Task Manager App

A simple and efficient **Task Manager** web application to create, manage, and track tasks with a clean and modern UI.

---

## 🚀 Features

- ✅ Create new tasks  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- 📌 Mark tasks as completed  
- 👥 Manage users (optional)  
- ⚙️ Settings panel  
- 🔔 Toast notifications for actions  

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
